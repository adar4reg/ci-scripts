    Downloading and Running GTS and CTS

    All binaries are provided under the terms of the Google MADA. The GMS Test Suite (GTS) and Compatibility Test Suite (CTS) are required to submit a build for launch approval.

    CTS and GTS currently support 64-bit Linux and Mac OS host machines. CTS and GTS will not work on Windows OS.

    Downloading CTS

    For information on downloading and running CTS, refer to Compatibility Test Suite on source.android.com.

    Running CTS/GTS using local Media files

    Refer to Run CTS/GTS media tests locally for the updated instructions on how to run CTS and GTS using local media contents. This will make test suites use local media files, instead of downloading them from Google server.

    Running CTS/GTS using MTT

    Multi-Test Transport (MTT) is a testing tool that Android developers and test engineers can use to employ a user interface for running standard Android test suites. It works with the Android Compatibility Test Suite (CTS) and with GTS, the Google Mobile Services (GMS) variant. 


    MTT is currently in pilot phase. Fill out the questionnaire if you are interested in piloting MTT.
